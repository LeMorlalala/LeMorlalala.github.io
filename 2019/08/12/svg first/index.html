如何围绕中心点旋转

svg 中 rotate 表示旋转，当只设置 rotate 时，可能得到的图形并不是想象中的样子，那是因为 svg 坐标是先旋转坐标系统，再根据位置对图形进行定位

会出现上图的位置时因为坐标系统以 0,0 为中心点先旋转了 45 度，然后再根据旋转之后的坐标系统，设置 x 和 y 固定图形位置，当我们希望图形绕着固定的点旋转的时候，有两种解决方法：

  1. transform设置translate(x,y)rotate(45)translate(-x,-y)即可围绕 x,y 旋转

  2. transform设置 rotate(45,x,y)也可以围绕 x,y 旋转

如何围绕中心点缩放

椭圆弧的路径

椭圆弧的命令以 A(绝对)或者 a(相对)，下面是时常记不住的 7 个参数们：

  1. 点所在椭圆的 x 半径和 y 半径
  2. 椭圆的 x 轴旋转角度 x-axis-rotation
  3. 需要圆弧的角度小于 180 度则为 0，表示小圆弧。圆弧角度大于或等于 180，则为 1
  4. 表示 start 点到 end 点的指向，逆时针则为 0，顺时针则为 1
  5. 终点的 x 坐标和 y 坐标

下面是一个应用圆弧的 demo:

我们不能使用一个路径来绘制一个完整的椭圆，如果弧形的起点和终点相同，则有无数方式定位椭圆。SVG 阅读器会跳过这样的圆弧命令。如果指定的椭圆半径太小导致不能覆盖起点和终点，则 SVG 阅读器会扩大椭圆直到它足够覆盖起点和终点。

贝塞尔曲线

二次贝塞尔

q 命令：Q 表示绝对路径，q 表示相对路径，命令后面紧跟两组坐标，分别是控制点和终点

上述表示的是从（30,75）到（300,120）的一条曲线，控制点在（240,30），画图的大致流程看下图

t 命令：包括 T t，是 svg 提供的流畅的二次曲线命令。这个命令后面紧跟的是曲线的下一个端点，控制点会自动计算，方法是”使新的控制点与上一条命令中的可控制点相对于当前中心对称”,从数学角度来讲，若控制点是 x1,y1 上一个线段的终点是 x,y 则新的控制点是

x2 = x+(x-x1) = 2x - x1; y2 = y+(y-y1) = 2y - y1

三次贝塞尔

c 命令：c 命令后面紧跟三组坐标，指定起点的控制点，终点的控制点，和终点。

s命令：表示平滑的三次贝塞尔曲线，新的曲线会把上一条曲线的端点作为它的起点，并且它第一个控制点是上一个终点控制点的中心对称点，我们只需要提供曲线的下一个端点控制点和端点

markker 元素

我们通常把<marker>放在<defs>中，表示可复用元素。它有以下几个常用属性:

refX="5" refY="5":指定哪个坐标与路径开始坐标对齐;
orient="auto"：表示标记自动旋转来匹配路径的方向。

注意：marker-mid会附加给除路径起点和终点之外的每个顶点，marker-start表示起点，marker-end表示终点
