工作原理

我们用d3.select()方法来实现对单个元素的选取，该select方法接受一个CSS3选择器字符串或者待操作对象的引用，并返回一个D3选集，随后，就可以用级联修饰函数对该选集的属性以及HTML进行操作了。

如果发现多个匹配元素，最终只返回第一个匹配的元素。

下面列出本书中用到的部分常见修饰函数。

  • selection.attr函数：用来读取或改变元素上的给定属性。

  • selection.style函数：用来给选定元素添加指定样式

  • selection.html函数：用来更改元素内的HTML

  • text同上，拥有三种方法修改文本内容

这些修饰函数可用于单个元素以及多个元素，当应用于多元素选集时，这些函数会依次作用于其中每个元素

当函数被作为参数传入修饰函数时，其实同时还有一些其他的隐含参数传入，最终实现了数据驱动计算，D3的强大之处就在于数据驱动的方式，它的名称数据驱动文档(data-driven document)，也正是来自于此，后续会讨论。
